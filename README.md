# roles-management

### Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### App uses JSON Server for fake API so it needs to be installed
```
yarn add json-server
```

### In the root folder of the app you will find db.json file, so to run server type:
```
json-server db.json
```

### Technologies used
- Vue CLI
- Vue
- Vue Router
- Vuex
- Axios
- JSON Server
